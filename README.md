# chunk-array

[![npm version](https://img.shields.io/npm/v/chunk-array.svg)](https://www.npmjs.com/package/chunk-array)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Array utilities — chunk, flatten, unique, shuffle, groupBy

## Installation

```bash
npm install chunk-array
```

## Functions

- `chunk` — Split array into chunks of given size
- `flatten` — Flatten nested arrays one level deep
- `unique` — Remove duplicates (by reference for objects)
- `shuffle` — Fisher-Yates shuffle (returns new array)
- `groupBy` — Group array items by key function
